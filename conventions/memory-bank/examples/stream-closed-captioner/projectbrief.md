# Project Brief

## Goal
Complete the Vite conversion for the stream-closed-captioner-extension project.

## Scope
- Migrate build/dev tooling to Vite.
- Preserve existing behavior, tests, and accessibility requirements.
- Keep changes minimal and aligned with current architecture.

## Out of Scope
- Feature changes unrelated to the Vite conversion.
- Major refactors not required for build compatibility.
