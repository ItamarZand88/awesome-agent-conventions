# Code of conduct

This project is a small research catalog. It works best when people bring
evidence, curiosity, and a willingness to label uncertainty honestly.

## Expected behavior

- Be respectful and constructive, especially when correcting maturity tiers or
  adoption claims.
- Critique sources and wording, not people.
- Prefer links, reproducible checks, and primary sources over vibes.
- Respect maintainers' decisions on scope, licensing risk, and whether a source
  should be vendored or only linked.

## Unacceptable behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Posting secrets, private data, or non-public material.
- Misrepresenting adoption evidence, source provenance, or licensing status.
- Repeatedly pushing off-scope entries after a maintainer decision.

## Reporting

For public, non-sensitive conduct concerns, open an issue and keep it factual.
For private safety or harassment reports, contact the maintainer through the
GitHub profile linked from the repository owner. Security-sensitive reports
should follow [SECURITY.md](SECURITY.md).

Maintainers may edit, hide, or remove comments and contributions that violate
this code of conduct, and may block participants when needed to keep the project
usable.
